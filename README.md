# Train Ticket System (JavaFX)

A complete Java-based train ticket reservation and management system with full UI integration and backend validation. Built using **JavaFX**, this project demonstrates a professional implementation of ticket booking, customer support, navigation logic, and product catalog features.

---

## 🚀 Features
- Train ticket booking and reservation
- Customer support messaging module
- Navigation and controller-driven UI actions
- Product catalog integration
- Polished UI with gradients and color accents
- Backend validation for reliable operations

---

## 🛠️ Technologies
- **Java** (Core logic)
- **JavaFX** (User Interface)
- **Maven/Gradle** (Build management)
- **FXML** (UI layout)

---

## 📂 Project Structure
- `src/` → Source code
- `resources/` → FXML layouts, CSS, and assets
- `bin/` / `target/` → Build outputs (ignored via `.gitignore`)
